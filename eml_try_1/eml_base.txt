Фундаментальное определение (Аксиома)Базовый оператор Exponent Minus Logarithm (EML) принимает два аргумента и вычисляется по формуле:$$eml(x, y) = e^x - \ln(y)$$Главное правило системы: В качестве стартовых "кирпичиков" разрешено использовать только константу 1 (или входную переменную $x$, если мы строим функцию). Все остальное должно выводиться через рекурсивное вложение $eml$.2. Вывод базовых константЧтобы алгоритм (или ИИ) понял, как строится базис, ему нужно показать, как добыть ключевые числа из единиц.Получение экспоненты ($e$):$$eml(1, 1) = e^1 - \ln(1) = e - 0 = e$$Получение $e^e$ (промежуточный шаг):Подставляем полученное $e$ в первый аргумент.$$eml(e, 1) = e^e - \ln(1) = e^e$$Получение нуля ($0$):Подставляем $e^e$ во второй аргумент оператора, где первый аргумент — единица.$$eml(1, e^e) = e^1 - \ln(e^e) = e - e = 0$$Полная запись нуля только через единицы:$$0 = eml(1, eml(eml(1, 1), 1))$$3. Вывод базовых функций (Символьная регрессия)Это ключевой раздел для машинного обучения и поиска аналитических формул. Показывает, как $eml$ симулирует другие математические операторы.Функция экспоненты ($e^x$):$$eml(x, 1) = e^x - \ln(1) = e^x$$Функция натурального логарифма ($\ln(x)$):Это самое важное доказательство работоспособности базиса. Требует трех шагов вложения:Шаг 1:$$eml(1, x) = e - \ln(x)$$Шаг 2 (оборачиваем в экспоненту):$$eml(eml(1, x), 1) = e^{e - \ln(x)} = \frac{e^e}{e^{\ln(x)}} = \frac{e^e}{x}$$Шаг 3 (оборачиваем в логарифм с вычитанием из $e$):$$eml(1, \frac{e^e}{x}) = e^1 - \ln\left(\frac{e^e}{x}\right) = e - (e - \ln(x)) = \ln(x)$$Полная запись функции натурального логарифма:$$\ln(x) = eml(1, eml(eml(1, x), 1))$$4. Выход в комплексную плоскость (Тригонометрия и $\pi$)Система неизбежно выходит в комплексную плоскость, если на вход второго аргумента (логарифма) попадает отрицательное число. Это позволяет извлекать геометрические константы.Используется главная ветвь комплексного логарифма: $\ln(-x) = \ln(x) + i\pi$.Генерация мнимой единицы и $\pi$:Если алгоритм в процессе мутаций (или направленного поиска) генерирует $-1$, вычисление $eml(0, -1)$ дает прямой выход на комплексные числа:$$eml(0, -1) = e^0 - \ln(-1) = 1 - i\pi$$Благодаря этому свойству, грамматика вида S -> 1 | x | eml(S, S) способна аппроксимировать синусы и косинусы через формулу Эйлера ($e^{ix} = \cos(x) + i\sin(x)$), так как $eml$ умеет генерировать как экспоненту, так и мнимые компоненты.
1. Врата в тригонометрию: Комплексный логарифмЧтобы базис EML мог генерировать периодические функции (синусы и косинусы), алгоритм должен уметь работать с отрицательными числами под логарифмом.В математике используется главная ветвь комплексного логарифма:$$\ln(-x) = \ln(x) + i\pi$$Для базовой отрицательной единицы ($-1$) логарифм выдает чисто мнимое число:$$\ln(-1) = i\pi$$Это дает нам фундаментальный мост к мнимой единице ($i$) и константе $\pi$. Например:$$eml(0, -1) = e^0 - \ln(-1) = 1 - i\pi$$2. Базовый осциллятор (Формула Эйлера в EML)Как только в системе появляется мнимая единица $i$, EML превращается в идеальный генератор волн.Мы знаем, что первый аргумент EML работает как чистая экспонента:$$eml(z, 1) = e^z$$Если на вход первого аргумента подать чисто мнимое число $ix$ (где $x$ — переменная, например, время или угол), оператор автоматически раскладывается на тригонометрические компоненты по формуле Эйлера:$$eml(ix, 1) = e^{ix} = \cos(x) + i\sin(x)$$Вывод для алгоритма:Вам не нужно собирать отдельную сложную структуру eml для $\sin(x)$ и $\cos(x)$. Достаточно подать $ix$ в левый аргумент $eml(ix, 1)$.Действительная часть результата $\Re(eml(ix, 1))$ — это $\cos(x)$.Мнимая часть результата $\Im(eml(ix, 1))$ — это $\sin(x)$.3. Классическая запись Синуса и Косинуса (через EML)Если архитектура требует строго аналитического выражения синуса и косинуса через экспоненты (чтобы оставаться в логике EML-графа), используются прямые следствия из Эйлера:Косинус:$$\cos(x) = \frac{e^{ix} + e^{-ix}}{2}$$В терминах EML (свободные константы типа деления на $2$ оставляем как масштабирующие веса $C=0.5$):$$\cos(x) = C \cdot [eml(ix, 1) + eml(-ix, 1)]$$Синус:$$\sin(x) = \frac{e^{ix} - e^{-ix}}{2i}$$В терминах EML (где константа $C = \frac{1}{2i}$):$$\sin(x) = C \cdot [eml(ix, 1) - eml(-ix, 1)]$$4. Гиперболическая тригонометрия (Бонус)Гиперболические функции ($\sinh, \cosh$) выводятся еще проще, так как не требуют выхода в комплексную плоскость (никаких $i$). Они крайне полезны в нейросетях как функции активации.Гиперболический косинус:$$\cosh(x) = \frac{e^x + e^{-x}}{2}$$В терминах EML:$$\cosh(x) = 0.5 \cdot [eml(x, 1) + eml(-x, 1)]$$Гиперболический синус:$$\sinh(x) = \frac{e^x - e^{-x}}{2}$$В терминах EML:$$\sinh(x) = 0.5 \cdot [eml(x, 1) - eml(-x, 1)]$$
