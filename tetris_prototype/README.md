# tetris_prototype — ранний прототип (v1, frame-by-frame)

Первая версия Тетриса с покадровым (frame-by-frame) управлением: PPO-оракул выбирал действие на каждом кадре. За 4.2M шагов агент так и не научился стабильно собирать линии — награда за линию размазывалась по сотням кадров.

Проект заменён актуальной версией в [`../tetris`](../tetris/) (placement-based action space: одно действие = одна постановка фигуры). Папка оставлена для истории и в дипломе фигурирует как «реализовавшийся риск v1»; не развивается.
